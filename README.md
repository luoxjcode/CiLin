# 语林 (YuLin)

这是一个 Android 项目。

## 项目结构

- `app/`: Android 应用模块
- `gradle/`: Gradle 相关配置
- `build.gradle.kts`: 根项目构建脚本
- `settings.gradle.kts`: 项目设置脚本

## 开发环境

- Android Studio: 建议使用最新稳定版
- Gradle: 8.7
- JDK: 17+

## 运行方式

1. 克隆项目到本地
2. 使用 Android Studio 打开项目
3. 等待 Gradle 同步完成
4. 选择设备并点击运行

## 忽略文件

项目包含 `.gitignore` 文件，配置了常见的 Android 开发忽略项，包括：
- Gradle 临时文件
- 构建产物
- 本地属性文件 (`local.properties`)
- IDE 缓存
